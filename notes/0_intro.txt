Basic I/O System (BIOS)
* BIOS chip on motherboard (like EEPROM)
* right after power on: OS at boot partition on HDD, need to move to RAM
* boot sequence: where BIOS looks for OS first
* BIOS load bootloader into RAM, CPU executes bootloader loaded into RAM

Executing multiple processes with a multicore CPU
* n-core CPU can only run n processes in parallel
* context switching to simulate running more in parallel
* clock interrupt every 50ms: suspends running process, OS decides what process to execute next, continue
